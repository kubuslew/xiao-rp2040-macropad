# Custom Macropad – XIAO RP2040 Based

## Overview
Custom-designed mechanical macropad built around the Seeed Studio XIAO RP2040 microcontroller.

Project includes schematic design, PCB layout, component selection and manual assembly.

Firmware implementation is planned as a next development stage.

## Hardware Specifications

- MCU: Seeed Studio XIAO RP2040
- 9 × Kailh mechanical switches
- 9 × PG15110S11 hotswap sockets
- 1 × PEC11R-4315F-S0012 rotary encoder
- 1 × WINSTAR WEA012832FWPP3N00000 OLED display
- USB-C powered

## Design Process

- Schematic design
- PCB layout optimization
- Routing of switch matrix
- Rotary encoder integration
- OLED display connection (I2C)
- Component footprint verification
- Manual soldering and inspection

## Assembly

- THT and SMD soldering
- Switch socket installation
- Encoder alignment
- Visual inspection and continuity testing

## Testing

- Power rail verification (3.3V)
- Short-circuit inspection
- Signal continuity tests
